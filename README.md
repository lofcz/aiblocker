## Getting Started

Clone the repository, install dependencies:
```ps
npm i
```

To build `zip` files ready for Chrome & Firefox Extension Stores, run:
```ps
npm run ship
```

This creates a `dist` folder with `chrome.zip` and `firefox.zip`.
